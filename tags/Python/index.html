随着python的崛起，Java是否会在未来几年没落

May 29, 2018
资讯

好歹是软件工程专业的，也算是科班学生，是不是对自己要求太低了？怎么还担心这种问题？编程的语言从来都只是皮毛而已。关键是你要想清楚自己要从事哪个方向？少年，要学会透过现象看本质啊。

你的问题也提到了大数据和人工智能的潮流，所以你的本质问题不是Python还是Java，而是你要去做后端，还是人工智能，亦或是大数据。人呐，想清楚自己想干什么是很重要的。

我可以很负责任的告诉你，如果做后端的话，Python不够。如果去做人工智能，也就是说所谓的调包工程师，Python勉强够用。如果你去做大数据Python也不够用。后端的主流是Java和Cpp。大数据也是用Java居多。所以你掂量掂量。

Python这几年是很火。因为到目前为止，没有比它更好的脚本语言。同时Python，特别适合做胶水语言。连接各种系统，处理数据，分析数据，什么都能干，上手难度低。同时机器学习生态圈对Python很友好，大把的框架都是Python写的。
