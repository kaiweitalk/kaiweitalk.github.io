2018 年 Java Web 开发和 Android 开发哪个更有前景

May 29, 2018
咨询

无论是Java web的开发，还是Android开发，都需要会Java，所以你可以先把Java基础，进阶部分都学了。

如果你有扎实的Java基础，再去学Java web的或者Android开发，其实都不是什么难事。

既然提主限定了时间维度是18年，那我肯定还是推荐学Java web的，后端开发不限于Java web，但是Java web是最常见的后端开发。15年的时候，客户端开发确实火，火到什么程度呢？一个培训班出来的，工作经验不到一年就可以月薪2.5万以上。

如果是在当时比较火的P2P互金或者O2O，那么薪资更高。很多人也是抓住了当时的机会，转了管理。有的人甚至能带一个技术部门，说实话让客户端或者前端来做tech leader我是不放心的。

但是15，16两年确实产生了很多客户端程序员。所以就现在的行情看，客户端的程序员已经有点供大于求了。

以前我在百度工作。我们大部门大概250个后端。还有20个前端。十个客户端。这个比例你可以参考一下，无论哪家公司，还是后端的需求更大。这十个客户端里边一半是iOS一半是Android。

后端的技术肯定比客户端的深，也更宽。一般来说，公司的技术总监或者高管，都是做后端出身的居多。

话虽这么说，兴趣能让你走得更长远，如果你对客户端开发更有兴趣，就要坚持去学去做客户端。我北大的同学里边，按大家理解的，应该都去做算法，或者去做后端。但是偏偏有那么几个同学，特别喜欢前端，毕业一干就是几年。而且干得特别开心。

最后，写客户端的和写前端的，往往都长得比较帅，比较容易找到女朋友。相比之下，写后端的往往都比较猥琐比较挫。嗯，就是这样的。

随着python的崛起，Java是否会在未来几年没落

May 29, 2018
咨询

好歹是软件工程专业的，也算是科班学生，是不是对自己要求太低了？怎么还担心这种问题？编程的语言从来都只是皮毛而已。关键是你要想清楚自己要从事哪个方向？少年，要学会透过现象看本质啊。

你的问题也提到了大数据和人工智能的潮流，所以你的本质问题不是Python还是Java，而是你要去做后端，还是人工智能，亦或是大数据。人呐，想清楚自己想干什么是很重要的。

我可以很负责任的告诉你，如果做后端的话，Python不够。如果去做人工智能，也就是说所谓的调包工程师，Python勉强够用。如果你去做大数据Python也不够用。后端的主流是Java和Cpp。大数据也是用Java居多。所以你掂量掂量。

Python这几年是很火。因为到目前为止，没有比它更好的脚本语言。同时Python，特别适合做胶水语言。连接各种系统，处理数据，分析数据，什么都能干，上手难度低。同时机器学习生态圈对Python很友好，大把的框架都是Python写的。

加上现在自媒体，培训班的宣传。搞得好像不学Python就要死了一样。实际上，如果你有其他语言的编程基础，再学python，跟喝杯白开水一样容易。

就现在工业界的情况来看Java是越来越牛逼。大公司里面像阿里，美团，全部都是Java系。创业公司里面用java的就更多了。Java的生态很好，开源丰富，性能优秀。适合任何公司，在任何时候用。如何这个公司不用Java，一定是早期技术领导不擅长Java。我实在想不出什么理由不用Java，除非你是写一个对内存，对性能要求极致的程序。从企业的角度看，Java也好招人。既然企业对Java的需求这么大，你还有什么理由不学Java？

对这个事定个结论。学生必须要懂一门强类型面向对象语言。Cpp和Java是必须要学好的。不用质疑，相信我得永生。
